# Приложение - список желаний  

## Основные сущности:  
- ### *Желание* 
    Включает в себя название, описание, ссылку, цену.  
    Обязательное поле только название.  

    Возможные действия:
    - Стандартный CRUD  
    - Возможность поделиться
- ### *Список желаний*  
    Список из желаний.  
    Имеет название
